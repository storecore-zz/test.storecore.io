Introducing StoreCore

Tired of all those software updates or a slow webshop? Keep on reading! We introduce to you: StoreCore, the first open-source e-commerce platform built by, with, and for you. We are building not only one of the fastest e-commerce platform that there is today (loads in just half a second), but we also put mobile first. Yes, from now on you can run your online store from the palm of your hand. We at StoreCore are therefore proud to present to you a dedicated e-commerce platform that grows with you. And hopefully grows on you.

smartphone Mobile first

Why not start with mobile? Everyone is using their phones today for almost everything. Now you can use your phone to run your business. Anyplace, anytime, and anywhere.

whatshot High performance

We are doing everything to set our goals and standards high. Performance is one of the main goals: your StoreCore web shop will be among the fastest sites in your business. StoreCore has a native full-page cache and supports Google AMP (Accelerated Mobile Pages).

security Security by design

StoreCore is built with security in mind. StoreCore is PCI DSS compliant and includes security features like whitelists, blacklists, and military-grade password encryption. To emphasize our dedication to security, all StoreCore hosting plans include a free SSL certificate.

shopping_cart Dedicated commerce

StoreCore is no some overgrown blog tool or yet another CMS. It’s a dedicated e-commerce platform. Built by, for, and with e-commerce and retail professionals.

business Multi-store support

StoreCore allows you to run up to 255 different web shops. StoreCore supports all world currencies, international shipping, and multiple languages. Currently English, French, German, and Dutch are supported.

local_florist Evergreen

Just like modern web browsers, StoreCore is an evergreen software framework: security patches and other software updates are downloaded and installed automatically.

build Scalable

StoreCore is designed as a framework for building web shops and online storefronts as well as e-commerce and retail applications.

free_breakfast Freedom

No more vendor lock-ins! StoreCore is free and open-source software (FOSS), published under the GNU General Public License. There is no proprietary or hidden software code: all open-source code is fully accessible and adjustable. And what’s more: all your business data is yours. Of course.
